确认框 (flyer.confirm) 示例 

//声明一个公共函数
function result(elm, value) {
    if (value) {
        flyer.alert("你点击了<span style='color:#009688'>[" + elm.innerHTML + "]</span>按钮.");
    } else {
        flyer.alert("你点击了<span style='color:#FF5722'>[" + elm.innerHTML + "]</span>按钮.");
    }
}

//简单询问
$("#confirm1").on("click ", function() {
    flyer.confirm("确定要这样做吗?", result);
});

//对话询问
$("#confirm2").on("click ", function() {
    flyer.confirm("确定要这样做吗?", result, {
        title: "询问框",
        isModal: true
    });
});

//来个有图标的
$("#confirm3").on("click ", function() {
    var strText = "<i class='fa fa-hand-peace-o flyer-icons flyer-warning'></i><span>其实这五种实现方式都是调用了一个类似叫基类的原型 dialog,这五种只是作者自已为了方便使用者写的几个方法作为快捷入口集成在代码里面而已，如果你足够熟悉，你还可以扩展出更多的快捷入口，作者很看好你哦!</span>";
    flyer.confirm(strText, result, {
        title: "<i class='fa fa-info-circle flyer-icons flyer-warning'><i><span>询问框</span>",
        isModal: true,
        area: [300, 200]
    });
});

//来个可以自定义按钮的看看
$("#confirm4").on("click ", function() {
    var strText = "<i class='fa fa-hand-peace-o flyer-icons flyer-warning'></i><span>其实这五种实现方式都是调用了一个类似叫基类的原型 dialog,这五种只是作者自已为了方便使用者写的几个方法作为快捷入口集成在代码里面而已，如果你足够熟悉，你还可以扩展出更多的快捷入口，作者很看好你哦!</span>";
    flyer.confirm(strText, null, {
        title: "<i class='fa fa-info-circle flyer-icons flyer-warning'></i><span>询问框</span>",
        isModal: true,
        area: [300, 200],
        btns: [{
            text: "我要得到这个按钮的文本",
            skin: "flyer-btn-warning",
            click: function(elm) {
                result.call(this, elm, true);
                this.close();
            }
        }, {
            text: "关闭",
            click: function(elm) {
                result.call(this, elm, false);
                this.close();
            }
        }]
    });
});

//敢不敢再加个按钮？
$("#confirm5").on("click ", function() {
    var strText = "<i class='fa fa-hand-peace-o flyer-icons flyer-warning'></i><span>这个就有点鸡肋了，因为这个其实本身已经脱离confirm的本质了，上述五种任何都可以实例化这样的弹框出来，你可以根据你的业务需求封装一个快捷入口出来，名字叫阿猫阿狗快捷入口都可以的!</span>";
    flyer.confirm(strText, null, {
        title: "<i class='fa fa-info-circle flyer-icons flyer-warning'></i><span>询问框</span>",
        isModal: true,
        area: [300, 200],
        btns: [{
            text: "不关闭按钮一",
            skin: "flyer-btn-warning",
            click: function(elm) {
                result.call(this, elm, true);
            }
        }, {
            text: "不关闭按钮二",
            skin: "flyer-btn-pink",
            click: function(elm) {
                result.call(this, elm, true);
            }
        }, {
            text: "关闭所有",
            skin: "flyer-btn-danger",
            click: function(elm) {
                flyer.closeAll();
            }
        }]
    });
});
